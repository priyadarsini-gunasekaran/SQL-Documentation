Introduction

SQL (Structured Query Language) is used to manage relational databases.

It allows users to create, read, update, and delete data in a structured way.

SQL is widely used in web applications, analytics, and data storage.

SELECT * FROM employees;
SQL Syntax

SQL follows a structured syntax for queries.

Each statement ends with a semicolon (;).

  • SELECT - Retrieves data
  • INSERT - Adds new data
  • UPDATE - Modifies existing data
  • DELETE - Removes data

Different SQL databases might have small syntax variations.

SELECT name FROM users WHERE age > 25;
Data Types

SQL supports various data types including:

  • INT - Integer values
  • VARCHAR - Variable-length text
  • BOOLEAN - True/False values
  • DATE - Stores date values

Choosing the correct data type optimizes storage and performance.

CREATE TABLE users (id INT, name VARCHAR(50));
CRUD Operations

CRUD stands for Create, Read, Update, Delete.

These operations form the foundation of SQL.

They are used to interact with a database.

INSERT INTO users (name) VALUES ('Alice');
Joins

Joins allow combining data from multiple tables.

Types of joins include:

  • INNER JOIN
  • LEFT JOIN
  • RIGHT JOIN
  • FULL OUTER JOIN

Joins are useful for combining related information.

SELECT orders.id, customers.name FROM orders 
INNER JOIN customers ON orders.customer_id = customers.id;
